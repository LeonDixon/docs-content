## Prerequisites

- Vue 3 or Nuxt 3: If you're using Vue 2 or Nuxt 2 you can use [VueFormulate](https://vueformulate.com).
- Node.js: 14.18.0, 16.12.0, or higher.
- Terminal: To run npm/yarn commands.
